## Installation
```bash
$ git clone git@github.com:yaru22/plaid-take-home.git
$ cd plaid-take-home
$ npm install
$ npm install -g nodemon
$ npm run db:reset
```

## Run
```bash
$ npm run start
```

## Test
```bash
$ npm run test
```
